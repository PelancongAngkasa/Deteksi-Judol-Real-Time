# 🎉 v2.0 Feature Summary

## 📊 Auto Scan 10 Menit Feature

### Apa Itu?
Sistem otomatis yang menjalankan 6 kali scanning URL dalam periode 10 menit dengan interval 2 menit untuk monitoring berkelanjutan.

### Timeline Scanning
```
├─ Menit 0-2   → Scan #1 (Website 1-5)
├─ Menit 2-4   → Scan #2 (Website 1-5)
├─ Menit 4-6   → Scan #3 (Website 1-5)
├─ Menit 6-8   → Scan #4 (Website 1-5)
├─ Menit 8-10  → Scan #5 (Website 1-5)
└─ Complete! ✓
```

### Real-time Metrics
Selama 10 menit, Anda bisa melihat:
- ⏱️ **Countdown Timer** - Sisa waktu monitoring
- 📊 **Scan Progress** - Scan #1/6, #2/6, dst
- ⚠️ **Live Detection Count** - Website terdeteksi real-time
- 📈 **Time Series Graph** - Grafik deteksi vs waktu
- 📋 **Hourly Summary** - Ringkasan per jam

## 📈 Grafik Deteksi Per Jam

### Y-axis: Jumlah Website Terdeteksi
```
Skala dinamis dari 0 sampai max detected count
Contoh:
  5  │     ●
     │    ╱ ╲
  3  │   ●   ●
     │  ╱     ╲
  1  │        ●
     └────────────
```

### X-axis: Waktu Scanning
```
Jam dalam format HH:MM:SS
Setiap scan memberikan 1 data point
Contoh:
10:00 → 10:02 → 10:04 → 10:06 → 10:08 → 10:10
 Scan#1  Scan#2  Scan#3  Scan#4  Scan#5  Scan#6
```

### Interpretasi Grafik

| Pattern | Arti | Action |
|---------|------|--------|
| 📈 Naik terus | Deteksi increasing | ⚠️ Alert needed |
| 📉 Turun terus | Deteksi improving | ✓ Good trend |
| 📊 Stabil | Normal operation | ✓ OK |
| 🔺 Sudden peak | Sudden spike | 🔴 Investigate |

## ⚙️ Cara Menggunakan

### Quick Start: Auto Scan

```bash
# 1. Terminal
cd "Deteksi Judol Real Time"
streamlit run app.py

# 2. Browser (auto opens)
http://localhost:8501

# 3. Sidebar
Select: "Auto Scan (10 Menit)"
Load: "Load dari file"
Click: "▶️ Mulai Auto Scan"

# 4. Monitor
- Watch countdown timer
- Monitor real-time metrics
- View time-series graph
- Wait for completion (~10 min)
```

### View Historical Data

```bash
# Open Analytics
streamlit run pages_analytics.py

# Or in sidebar dropdown (if configured as multi-page)
- Select "Analytics" tab
- Pick time range: 24 hours, 48 hours, etc.
- View trends, export CSV
```

## 📝 Example Workflow

### Day 1: Setup & First Scan
```
1. Run setup.bat / bash setup.sh
2. streamlit run app.py
3. Load URLs
4. Click "Auto Scan 10 Menit"
5. Monitor dashboard (10 min)
6. View results → Export CSV
```

### Day 2-7: Continuous Monitoring
```
1. Run auto scan setiap jam
2. Data accumulates di scan_data/
3. View trends di Analytics
4. Generate weekly report
```

### Weekly Report
```
1. Open Analytics dashboard
2. Select: 168 hours (7 days)
3. View trends: Peak hours, daily patterns
4. Export: Full week data to CSV
5. Investigate: Any unusual patterns
```
